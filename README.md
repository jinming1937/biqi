# biqi

一个简单，可信任，可以控制的React UI库，这里集成开发、测试、发布

* 安装

```bash
 yarn install
```

* 运行

```bash
 yarn dev
```

* 构建

```bash
 yarn build
```

* 版本(/packages/ui下)

```bash
  npm version [update_type]
```

> * 主版本号[major]：当你做了不兼容的 API 修改
> * 次版本号[minor]：当你做了向下兼容的功能性新增
> * 修订号[patch]：当你做了向下兼容的问题修正。

* 发布(/packages/ui下)

```bash
  npm publish
```
