# React UI Biqi

一个简单，可信任，可以控制的React UI库

## 📦 Install

```bash
npm install @biqi/ui
```

```bash
yarn add @biqi/ui
```

## 🔨 Usage

导入UI库

```jsx
import {Input} from '@biqi/ui';

const App = () => (
  <>
    <Input value="??" />
  </>
);
```

导入样式：

```less
@import '../node_modeules/@biqi/theme/index.css';
```
